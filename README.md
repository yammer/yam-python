# Yampy

Yampy (“Yam Pie”) is the official Python library for interacting with
Yammer's API.

## Installation

Install with pip:

```sh
pip install yampy
```

Or with easy_install:

```sh
easy_install yampy
```

Or manually:

```sh
git clone git://github.com/yammer/yam-python.git
cd yam-python
python setup.py install
```
